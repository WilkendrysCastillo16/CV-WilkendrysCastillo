Proyectos

Bolsas de Empleos (Job.com)

El Proyecto consiste en un sitio web de una Bolsa de Empleos. Desarrollado en Angular y se utilizó Angular Material para diseñar algunas de sus interfaces. Por otro lado, utilizamos una API creada por nosotros con la tecnología de .NET CORE, la cual vinculamos a una base de datos SQL SERVER que subimos en conjunto a los servicios de Microsoft Azure. Con la existencia de tres tipos de usuarios, usuario normal, usuario póster y usuario administrador. El usuario normal solo puede ver los empleos y la descripción. El usuario póster puede agregar empleos. Mientras que el administrador puede hacer todo eso y el CRUD completo de empleos, usuarios, categoría, configuración. Entre algunos casos de usos más.

Cabe destacar que este proyecto se llevó a cabo de manera colaborativa con mis compañeros Elian Toribio Germosen, Junior Samuel de los Santos, José Miguel Upia y Keury Alberto Lendof. Utilizando git como control de versiones y GitHub.
